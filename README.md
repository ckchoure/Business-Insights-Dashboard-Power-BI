# 📊 Business Insights Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Business Insights Dashboard** developed using **Power BI** to monitor and analyze business performance across multiple departments.

The dashboard provides actionable insights into **Sales, Finance, Marketing, Supply Chain, and Executive Performance** using interactive visualizations and KPIs.

Data was extracted from **SQL Server**, transformed using **Power Query**, and analyzed using **DAX** measures.

---

# 🚀 Objectives

- Analyze business performance
- Track important KPIs
- Compare current performance with previous years and targets
- Enable data-driven decision making
- Create department-wise business reports
